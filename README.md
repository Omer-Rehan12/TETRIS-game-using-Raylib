🎮 Tetris Game (Raylib)

A classic Tetris game built using the Raylib graphics library. 
This project focuses on implementing core game development concepts such as real-time rendering, collision detection, 
and grid-based mechanics.  

Features Classic Tetris gameplay Piece generation and randomization Rotation system for all tetromino shapes Collision 
detection with boundaries and blocks Line clearing system with scoring Increasing difficulty over time Smooth real-time 
rendering using game loop

Tech Stack 
C / C++ Raylib Graphics Library 

What I Learned 
Game loop design and real-time updates Handling user input in games Grid-based movement systems Collision detection logic Managing game state efficiently 

How to Run 
Install Raylib on your system 
Clone this repository 
Compile the code using a C/C++ compiler linked with Raylib 
Run the executable
